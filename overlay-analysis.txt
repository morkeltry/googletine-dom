YouTube DOM Overlay Analysis
================================

Problem: Grey screen overlays content, but disappears when hovering DevTools

Findings:
---------
1. Consent dialog elements ARE in the HTML (not created by JS):
   - <ytd-consent-bump-v2-lightbox id="lightbox"> with z-index: 2202
   - <tp-yt-iron-overlay-backdrop opened="" class="opened"> with z-index: 2201

2. Content flashes up then disappears → JavaScript controlling visibility

3. DevTools interaction suggests:
   - Overlay responds to focus/window events
   - Possible DevTools detection
   - Clever pointer-events handling

Key Elements Found:
------------------
- Consent dialog: <ytd-consent-bump-v2-lightbox id="lightbox">
- Backdrop: <tp-yt-iron-overlay-backdrop opened="" class="opened" style="z-index: 2201;">
- Dialog: <tp-yt-paper-dialog id="dialog" style="z-index: 2202;">

// Auto-remove consent dialog when it appears
setInterval(() => {
  const backdrop = document.querySelector('tp-yt-iron-overlay-backdrop[opened]');
  const dialog = document.querySelector('ytd-consent-bump-v2-lightbox');

  if (backdrop) backdrop.remove();
  if (dialog) dialog.remove();
}, 100);

// Or use MutationObserver to catch dynamic creation
const observer = new MutationObserver((mutations) => {
  mutations.forEach((mutation) => {
    mutation.addedNodes.forEach((node) => {
      if (node.nodeType === 1) {
        if (node.tagName === 'TP-YT-IRON-OVERLAY-BACKDROP' ||
            node.tagName === 'YTD-CONSENT-BUMP-V2-LIGHTBOX') {
          node.remove();
        }
        // Also check children
        const found = node.querySelector?.('tp-yt-iron-overlay-backdrop, ytd-consent-bump-v2-lightbox');
        if (found) found.remove();
      }
    });
  });
});

observer.observe(document.body, { childList: true, subtree: true });

Solutions:
---------
1. Remove consent elements from HTML before serving (server-side)
2. Run cleanup JS after page load (client-side)
3. Forward cookies so browser has proper consent state (long-term)
4. Find and disable specific polymer script that creates/manages consent
